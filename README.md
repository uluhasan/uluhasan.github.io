# Kişisel Web Sitesi

Bu repository, kişisel web sitemi barındırmaktadır. Site, profesyonel özgeçmişimi ve iletişim bilgilerimi içeren basit ve şık bir tasarıma sahiptir.

## İçerik

- `index.html`: Ana web sayfası
- `index.css`: Sitenin stil dosyası
- `index.js`: JavaScript fonksiyonları
- `hasan_ulu_cv.pdf`: Türkçe CV
- `hasan_ulu_resume.pdf`: İngilizce CV
- `bayrak.svg`: Site içerisinde kullanılan bayrak ikonu

## Teknolojiler

- HTML5
- CSS3
- JavaScript

## Yerel Olarak Çalıştırma

1. Repository'yi klonlayın:
```bash
git clone https://github.com/uluhasan/uluhasan.github.io.git
```

2. İndirdiğiniz klasöre gidin:
```bash
cd uluhasan.github.io
```

3. `index.html` dosyasını bir web tarayıcısında açın.
